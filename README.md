# QuickNav MV3

把现有的 QuickNav 用户脚本封装为一个 Chrome MV3 扩展（ChatGPT / Gemini Enterprise / Genspark）。

## 安装

1. 打开 `chrome://extensions`
2. 开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本目录：`QuickNav-MV3`

## 使用

- 打开支持的网站页面后，右上角会出现 QuickNav 面板（与原脚本一致）
- 扩展不提供弹出菜单（popup）；所有操作都在页面内面板完成

## 注意

- 如果你还启用了 Tampermonkey 同名脚本，建议先禁用，避免重复注入。
