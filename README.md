# QuickNav MV3

把现有的 QuickNav 用户脚本封装为一个 Chrome MV3 扩展（ChatGPT / Gemini / Grok / Genspark / 文心一言 / DeepSeek / Qwen / GLM）。

## 安装

1. 打开 `chrome://extensions`
2. 开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本目录：`QuickNav-MV3`

## 使用

- 打开支持的网站页面后，右上角会出现 QuickNav 面板（与原脚本一致）
- 点击扩展图标，会打开一个简单信息面板：作者/版本 + 检查更新
  - 点击「配置」可进入设置页：控制在哪些网站启用 + 各网站默认 🔐（防自动滚动）（默认全锁）
  - 扩展「重新加载」后，已打开的页面通常需要刷新一次才会重新注入脚本

## 注意

- 如果你还启用了 Tampermonkey 同名脚本，建议先禁用，避免重复注入。
